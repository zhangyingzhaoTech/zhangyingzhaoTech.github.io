计算机使用经验杂谈

发表于 2021-09-04

磁盘管理

硬盘是计算机中存储数据的主要载体。现如今，硬盘主要分为两类，一类是机械硬盘，具有数据保存稳定、容易恢复、价格便宜等优点，但是也伴随着数据读取速度慢等缺点，适合用于数据的保存。另一类是固态硬盘，具有数据读取速度快的优点，但是数据恢复较难，且价格昂贵，一般适用于安装系统和应用软件。

现在的较新的电脑一般都有以上两个硬盘，比较推荐的是将系统和软件安装在固态硬盘中，数据保存在机械硬盘中，这样电脑的运行速度更快。机械硬盘和固态硬盘都需要分区，一般建议系统单独分区，容量应该50G以上较好，而应用软件也应该单独分区，容量在100G以上，安装软件时避免占用系统所在的硬盘空间。而机械硬盘的分区不做强制要求，按照自己的使用习惯进行分区即可。

文件管理

为了避免文件太多产生不必要的混乱，应当将一类文档归入文件夹中，将文件一层一层保存，但是不建议子文件夹的带数太多，不建议超过5代。

对于英语较好的用户，建议使用英文的文件名和计算机用户名，因为很多软件对中文支持较差，软件运行时系统可能无法正确地找到文件的路径。

内存清理

使用计算机时要及时将不用的文件转移到回收站中，并及时清理回收站。而对于系统自身产生的垃圾，可以使用系统自带的驱动器工具进行清理和优化，避免使用360卫士、鲁大师、电脑管家等流氓软件进行垃圾清理和优化。

同样，病毒的查杀也只需要使用系统自带的防火墙即可。

浏览器

强烈建议使用系统自带的Microsoft Edge、谷歌的Chrome浏览器，两者页面简洁，且几乎没有广告植入，另外推荐的是火狐浏览器Firefox。非常不建议使用其他浏览器，包括但是不限于**360浏览器**。

文档编辑器

对于办公需求的用户，特别是经常使用Word\PowerPoint\Excel等软件的客户，那么微软公司的推出的Microsoft Office套装无疑是最好的选择，但是升级版也是付费的，对于普通用户，基础班足够用。对于经济条件不好的用户，可以在一些网站上找到绿色版本可供使用，但是本人并不倡导使用盗版软件，应当积极支持正版软件。国内金山公司的WPS功能和MS Office类似，可供选择，但是此软件占用内存较大，可能造成电脑运行缓慢，并且广告较多。

对于只进行简单文字编辑的用户可使用记事本或者Markdown编辑文本，Markdown语法较为简单，现如今很多网站也支持Markdown类型的文本，Markdown本身也支持LaTeX公式编辑。推荐的Markdown编辑器有Typora，本文档便是使用Typora语法进行编辑的。

PDF阅读器

目前支持PDF文件阅读的软件较多，微软自带的Microsoft Edge浏览器便支持PDF阅读，且用户体验较好，推荐使用。比较推荐的轻量级的PDF软件有SumatraPDF，占用内存较少，且和Microsoft Edge一样都不是独占文件且免费的程序。对于有PDF编辑的用户，则推荐使用Adobe Acrobat DC进行PDF文件的查看和编辑，但是升级版是付费的。对于安装WPS套装的电脑，也可以使用此软件进行查看和编辑，但是升级版也是付费的。

文件解压缩软件

推使用WinRAR进行文件的压缩和解压文件，占用内存较小，缺点是此软件是付费软件，但同样在一些网站上可以找到绿色版本。

截图软件

截图直接使用系统自带的截图工具即可，在Windows附件中即可找到。另外，QQ软件貌似也支持截图。

其他

对于电脑硬盘容量较小的用户，可以使用网盘存储和备份数据，用户最多的网盘为百度网盘，很多资源也都是通过百度网盘进行分享。阿里云、蓝奏云也具有一定的用户基础。此外，现在很多手机厂商也推出自己的网盘工具供给用户免费使用，这些也可以作为一个选择。

大多数的软件的下载都可以在自身的官网进行下载，安装和使用也有相应的说明文档，仔细阅读动手操作即可。

最重要的事

通过观看某个系列的教学视频或者某本教程学习使用电脑不是一个很有效率的方法，比较推荐的是边学边用，遇到不会的问题上网查询，百度永远是最好的老师，实在不行向周围的大佬请教即可。

LaTeX学习笔记

发表于 2020-12-23 更新于 2021-09-04

插入图片

导入插入图片宏包
\usepackage{graphicx}
%导入图片浮动格式
\usepackage{float}

在想要插入图片的地方使用以下命令：

\begin{figure}[ht]
    \centering
    \includegraphics[scale=0.6]{fullscreen.png} %插入图片
    \caption{this is a figure demo} %图片标题和自动编号
    \label{fig:label} %图片标签，便于引用，引用图片 \ref{label}
\end{figure}

其中

  • h(here) : 在此处插入
  • t(top): 在页最上方插入
  • b(bottom): 将图形放置在页面的底部
  • p(page): 将图形放置在一只允许 有浮动对象的页面上
  • H: 将图片紧随文字插入

参数详解：

  • height: 图形高度
  • width: 图形宽度
  • scale: 图形整体大小缩放

最新Hexo+github搭建个人博客教程

发表于 2020-11-19 更新于 2020-11-20

安装nodejs

直接百度nodejs或者点击http://nodejs.org/进行下载，安装稳定版本。安装后打开命令行输入

node -v

检查是否安装成功。

Git 安装

打开百度搜索github for windows，进入官网http://gitforwindows.org/点击下载安装，命令行下输入git检验安装成功即可。

Hexo本地运行

打开本地文件夹，以安装在D盘为例，新建一个名为blog的文件夹。右击鼠标打开Git bash环境输入安装命令

npm install hexo-cli -g

回车，安装完成后，输入

hexo init zhangsan

回车完成初始化，上述zhangsan表示博客名（用户名）。进入博客目录，git bash命令行下输入

npm install

回车安装依赖。安装完成后直接输入

hexo s

回车，即可完成Hexo本地运行。在浏览器中输入 http://localhost:4000/即可显示运行结果。若成功跳转至Hexo主页，则表示环境配置成功。

Hexo部署到github

注册github账号

百度输入github选择官网http://github.com/即可注册。注意用户名会成为你的Hexo博客用户名。

接着添加仓库，注意仓库名 （Repository name) 要按照: username.github.io 的格式来，如我注册的用户名为zhangsan，那么我的仓库名 （Repository name) 为： zhangsan.github.io .

配置SSH KEY

打开git所在文件夹，右击鼠标，选择git bash exe进入git命令行，输入

ssh-keygen -t rsa -C "邮件地址"

然后连续3次回车，最终会生成一个文件在用户目录下，打开用户目录，找到.ssh\id_rsa.pub文件，记事本打开并复制里面的内容，打开你的github主页，进入个人设置 -> SSH and GPG keys ->New SSH key. Title可任意填写。输入

ssh -T git@github.com # 注意邮箱地址不用改

回车检测是否成功。

此时还需要进一步配置：

git config --global user.name "username" // 你的github用户名，非昵称   
git config --global user.email  "xxx@qq.com"// 填写你的github注册邮箱
