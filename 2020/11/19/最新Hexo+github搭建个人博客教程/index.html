安装nodejs

直接百度nodejs或者点击http://nodejs.org/进行下载，安装稳定版本。安装后打开命令行输入

检查是否安装成功。

注册github账号

百度输入github选择官网http://github.com/即可注册。注意用户名会成为你的Hexo博客用户名。

接着添加仓库，注意仓库名 （Repository name) 要按照: username.github.io 的格式来，如我注册的用户名为zhangsan，那么我的仓库名 （Repository name) 为： zhangsan.github.io .

Git 安装

打开百度搜索github for windows，进入官网http://gitforwindows.org/点击下载安装，命令行下输入git检验安装成功即可。
